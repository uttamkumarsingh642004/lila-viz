# LILA BLACK вЂ” Player Journey Visualizer

An interactive web tool for visualizing 5 days of gameplay telemetry from LILA BLACK, an extraction shooter. Built as a take-home assignment for Lila Games.

**Live demo:** [https://lila-viz.vercel.app](https://lila-viz.vercel.app)

## Setup

### Prerequisites
- Python 3.9+ with `pip`
- Node.js 18+

### 1. Clone and install

```bash
git clone https://github.com/yourusername/lila-viz.git
cd lila-viz
npm install
```

### 2. Prepare data

Place the raw data in the parent directory so the structure looks like:
```
lila-viz/          в†ђ this repo
../player_data/player_data/
    February_10/
    February_11/
    ...
    minimaps/
```

Install Python dependencies and run the preprocessor:
```bash
pip install pyarrow pandas numpy
python preprocess.py
```

This reads all 1,243 parquet files and outputs:
- `public/data/ambrosevalley.json`
- `public/data/grandrift.json`
- `public/data/lockdown.json`
- `public/data/metadata.json`

### 3. Copy minimaps

```bash
cp ../player_data/player_data/minimaps/* public/minimaps/
```

### 4. Run locally

```bash
npm run dev
# App available at http://localhost:5173
```

### 5. Build for production

```bash
npm run build
# Output in dist/
```

## Data Notes

- Source: 89,104 telemetry events from 796 matches across 3 maps, Feb 10вЂ“14, 2026
- Format: Apache Parquet files with `.nakama-0` extension
- Key schema columns: `user_id`, `match_id`, `map_id`, `x`, `y` (elevation), `z`, `ts`, `event`
- See `ARCHITECTURE.md` for coordinate system and timestamp handling details
